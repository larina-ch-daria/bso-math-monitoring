# Станция проверки (React)

React-станция чернового
выставления баллов по сканам рукописных работ (PDF) с проверкой человеком.

Ключевые решения:
- грейдинг-промпт запрещает 0 баллов, если ответ ученика частично совпадает с ключом;
- заметки формируются на клиенте («ты: … · верно: …»), а не моделью — это
  убирает галлюцинированные расхождения;
- `max_tokens` ≥ 2000 при множестве подзаданий — иначе обрезаются последние;
- экспорт — через модалку copy-to-clipboard
